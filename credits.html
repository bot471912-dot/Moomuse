Crédits du projet

Cette page rassemble les principaux crédits techniques, visuels et organisationnels liés au site vitrine Pyxelze et à sa présentation publique. Elle a pour but de rendre visibles les contributions du projet, les services utilisés et les dépendances majeures sur lesquelles repose l'interface.

Équipe et contributions

Le projet Pyxelze est présenté publiquement à travers une équipe mise en avant sur le site. Cette section peut s'appuyer sur des profils publics Discord pour afficher le nom, l'avatar et l'identité visuelle des personnes associées au projet.

Les rôles affichés sur le site ont une vocation descriptive et permettent d'identifier les domaines de contribution principaux, par exemple le développement, l'hébergement, la sécurité, l'audit ou l'accompagnement juridique.

Technologies du site

Le site utilise notamment Next.js pour le framework applicatif, React pour le rendu de l'interface, Tailwind CSS pour la couche de style et diverses bibliothèques d'interface pour la composition visuelle et l'expérience utilisateur.

Des services externes peuvent être sollicités pour certaines fonctions ciblées, comme l'affichage de profils Discord ou l'accès aux téléchargements publiés via GitHub Releases.

Données affichées et sources externes

Certaines données visibles sur le site proviennent de sources externes ou publiques, comme les avatars et noms d'affichage associés aux profils Discord de l'équipe lorsque l'API Discord est configurée côté serveur.

Les liens de téléchargement peuvent pointer vers des fichiers hébergés sur GitHub. Le site peut également mémoriser localement la langue choisie dans le navigateur afin d'améliorer le confort d'utilisation lors des visites suivantes.

Logos, visuels et contenus

Le logo Pyxelze, les éléments de présentation, les visuels du site et les textes rédigés pour la communication du projet restent associés au projet ou à leurs ayants droit respectifs, sauf mention contraire explicite.

Si un crédit complémentaire doit être ajouté pour un visuel, une ressource, une illustration, une capture, une bibliothèque ou une contribution spécifique, cette page peut être enrichie pour détailler plus précisément l'origine et les conditions d'utilisation de chaque élément.

Mise à jour et contact

La présente page de crédits peut évoluer à mesure que l'équipe, les services utilisés, les hébergements, les dépendances techniques ou les partenaires du projet changent.

En cas d'oubli, d'erreur de crédit ou de demande de précision, il est recommandé de faire valider la version finale de cette page par l'équipe du projet avant publication définitive.
